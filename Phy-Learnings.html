Project Overview

تطبيق تعليمي ذكي يدمج بين الأسئلة التفاعلية، الاختبارات، والمحاكاة العلمية لتسهيل فهم المفاهيم الفيزيائية للطلاب بمختلف المستويات.

Key Features

  • نظام أسئلة تفاعلي مع نتائج فورية
  • مستويات تعليمية (مبتدئ - متوسط - متقدم)
  • نظام نقاط ومكافآت لتحفيز التعلم
  • شروحات مبسطة للفيزياء مع رسوم توضيحية
  • تتبع تقدم الطالب وتحليل الأداء
  • إشعارات يومية للاختبارات
  • Dark Mode لتجربة مريحة

Technologies Used

  • Laravel API Backend
  • Flutter (Android & iOS)
  • Firebase Authentication
  • Cloud Firestore / MySQL
  • RESTful API Integration
  • Admin Dashboard (Laravel + Vue.js)

Project Impact
